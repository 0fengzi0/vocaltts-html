# vocaltts-html
vocaltts项目的前端开源地址,项目使用vue-cli构建

欢迎star

在线展示页面(旧版)

http://tts.5ixf.cc

##### (1)开始
###### github
``` bash
git clone https://github.com/0fengzi0/vocaltts-html.git
```
###### 码云(备用,更新较慢)
``` bash
git clone https://gitee.com/feng__zi/vocaltts-html.git
```

##### (2)开始构建
``` bash
npm build
```

##### (3)将生成的dist目录内的文件放到网站的根目录
